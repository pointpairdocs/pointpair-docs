## Installation

##### 1. Download the PointPair Payments for WooCommerce plugin from the Merchant Portal business or docs page
![title](images/woocommerce1.png)
##### 2. Navigate to the WordPress Admin Dashboard
##### 3. Navigate to the Plugins > Add New
![title](images/woocommerce2.jpeg)
##### 4. Click the 'Upload Plugin' button
![title](images/woocommerce3.png)
##### 5. Upload the PointPair-WooCommerce.zip file you downloaded in step 1
![title](images/woocommerce4.png)
##### 6. Click 'Install Now'
![title](images/woocommerce5.png)
##### 7. In the next page click 'Activate Plugin'
![title](images/woocommerce6.png)

## Configuration

##### 1. Navigate to: Plugins > Installed Plugins
##### 2. Locate the 'PointPair Payments for WooCommerce' in the plugin list and click the 'Settings' button
![title](images/woocommerce7.png)
##### 3. Go to the API Keys section in the plugin settings page
![title](images/woocommerce8.png)
##### 4. Head over to the PointPair Merchant Portal Dashboard to retrieve the API and Secret Keys
![title](images/woocommerce9.png)
##### 5. Add the Keys in their respective fields
![title](images/woocommerce10.png)
##### 6. Click the 'Save changes' button at the bottom of the page
##### 7. Congratulations PointPair is now live on your website!
